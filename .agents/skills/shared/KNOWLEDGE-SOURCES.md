# Organizational knowledge sources

Beyond code and the web, relevant context often lives in team
tools. Reach these through MCP servers and search them when
available:

- **Confluence** (Atlassian MCP) — pages, specs, design docs,
  runbooks. Full documents.
- **Google Drive** (Google Drive / Workspace MCP) — docs,
  sheets, slides. Search by topic keywords.
- **Slack** (Slack MCP) — not a document store, but
  conversations, decisions, and tribal knowledge. Search public
  channels and threads for relevant discussion. Cite useful
  messages with a permalink and quote the relevant snippet.

## When a source is unavailable

If an MCP server is unavailable, not installed, or misconfigured,
tell the user which source you couldn't reach, then continue with
the sources you do have. Don't block on a missing integration.
